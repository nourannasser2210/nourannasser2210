## ⚡ About Me

I'm a 4th-year Computer Science student at **Egyptian Chinese University** (graduating 2027), currently training in the **Data Engineer Track at Digital Egypt Pioneers Initiative (DEPI)** — but where I actually spend my nights is deep learning.

- 🔭 Currently building **emotion-aware AI systems** — text-based and vision-based
- 🌱 Deepening my skills in **Transformers, Transfer Learning, and MLOps**

### 🎭 [EmotiSense — Text Emotion Classifier](https://github.com/nourannasser2210/emotion-classification-nlp)

A BiLSTM deep learning model that reads a sentence and detects the emotion underneath it — anger, fear, joy, love, sadness, or surprise.

- **Architecture:** 2-layer BiLSTM (128 → 64 units)
- **Accuracy:** 91% on held-out test data
- **Data:** 16k train / 2k val / 2k test samples
- Model hosted on **Hugging Face Hub**, app deployed on **Hugging Face Spaces**

**Stack:** `TensorFlow` `Keras` `BiLSTM` `Streamlit` `Hugging Face`

🔗 [Live Demo](https://huggingface.co/spaces/nourannasser22/emotion-detector-nlp) · [Source Code](https://github.com/nourannasser2210/emotion-classification-nlp)
